# Social Media Writing Workflow

Use this workflow for platform-native short-form content.

## Goal

Produce concise, high-signal posts that are clear, engaging, and optimized for a professional audience.

## Process

1. Identify the platform and intended audience.
2. Choose one main idea only.
3. Start with a hook strong enough to earn the next line.
4. Expand with short sections, examples, or a narrative arc.
5. End with a clear takeaway, question, or CTA.

## Platform Guidance

### LinkedIn

- Use short paragraphs with strong first two lines
- Balance professionalism with a human voice
- Favor one insight + one example + one takeaway

### X / Twitter Threads

- Each post should carry one idea
- Open with a thesis or provocative takeaway
- Use progression: hook -> context -> insight -> examples -> close

## Output Guidance

- Save LinkedIn content to `./.mosswriter/linkedin/<slug>/post.md`
- Save X thread drafts to `./.mosswriter/twitter/<slug>/thread.md`
- If needed, save image direction to `./.mosswriter/<platform>/<slug>/image_prompt.md`

## Style

- Tight, energetic, and specific
- No vague motivational language
- No overstuffed hashtags
- Avoid sounding like a generic marketing bot
