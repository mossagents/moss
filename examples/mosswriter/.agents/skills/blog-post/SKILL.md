# Blog Post Workflow

Use this workflow for long-form written content.

## Goal

Produce a publishable blog post with a clear argument, strong structure, and actionable takeaways.

## Process

1. Clarify the target audience and desired outcome.
2. If the topic requires evidence or current examples, delegate research first.
3. Identify the core thesis before writing.
4. Create a strong outline before drafting.
5. Draft the post in full paragraphs, not bullet fragments.
6. Strengthen the introduction, transitions, and conclusion.
7. If useful, generate a cover image brief and save it alongside the post.

## Suggested Structure

1. Hook / why this matters now
2. Context or problem framing
3. Main argument or framework
4. Examples, evidence, or comparison
5. Practical takeaways
6. Conclusion / CTA

## Output Guidance

- Save the post to `./.mosswriter/blogs/<slug>/post.md`
- If needed, save research notes to `./.mosswriter/research/<slug>.md`
- If needed, save cover art guidance to `./.mosswriter/blogs/<slug>/cover_prompt.md`

## Style

- Prefer clarity over cleverness
- Use descriptive headings
- Keep paragraphs compact but substantive
- Avoid filler intros and generic AI tone
